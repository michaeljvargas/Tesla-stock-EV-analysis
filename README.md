# Tesla Stock Prediction & EV Market Analysis

## Project Overview
This project analyzes Tesla’s historical stock performance and develops machine learning models to predict next-day stock prices. The analysis also incorporates global EV market trends to provide business context and insights.

## Business Problem
Tesla operates in a highly volatile and competitive electric vehicle market. Investors and analysts need better tools to understand short-term stock price movements and identify patterns that can support data-driven decision-making.

## Objectives
- Predict Tesla’s next-day closing stock price using historical data
- Evaluate multiple machine learning models to determine the best-performing approach
- Analyze EV market trends to provide broader industry context

## Why This Matters
Accurate short-term stock predictions can:
- Help investors make informed trading decisions
- Improve understanding of market behavior and volatility
- Provide insight into how industry growth impacts company performance

## Data Sources
- Tesla Historical Stock Data (2010–2025)
- Global EV Market Dataset (2010–2024)

## Methodology (CRISP-DM)
1. Business Understanding  
2. Data Understanding  
3. Data Preparation  
4. Modeling (3 models tested)  
5. Evaluation  
6. Deployment (business recommendations)

## Models Used
- Linear Regression  
- Random Forest Regression  
- K-Nearest Neighbors (KNN)

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score

## Key Insights
- Tesla stock prices are strongly influenced by short-term trends and volatility
- Nonlinear models (Random Forest) perform better than linear models
- EV market growth provides strong supporting context for Tesla’s long-term performance
