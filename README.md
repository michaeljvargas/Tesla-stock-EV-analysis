# Tesla Stock Prediction Using Machine Learning

## Overview
This project analyzes Tesla’s historical stock performance and predicts next-day stock prices using machine learning models. It also incorporates electric vehicle (EV) market trends to provide business context and support insights.

## Business Problem
Tesla operates in a rapidly evolving EV market. Investors and analysts need better tools to understand short-term stock movements and identify patterns that can inform decision-making.

## Objective
The goal of this project is to:
- Predict Tesla’s next-day closing stock price
- Analyze key drivers such as volatility and trend indicators
- Provide insights into how EV market growth aligns with Tesla’s performance

## Data Sources
- Tesla Historical Stock Data (2010–2025)
- Global EV Market Data (2010–2024)

## Models Used
- Linear Regression
- Random Forest Regression
- K-Nearest Neighbors (KNN)

## Key Features
- Moving averages (MA_7, MA_30, MA_90)
- Daily returns and volatility
- Price change metrics
- Time-based features (month, quarter, day of week)

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- Google Colab
- GitHub
