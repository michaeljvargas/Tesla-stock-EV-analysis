# Data Folder

This folder contains all datasets used in the project.

## Files
- tesla_stock_data_2010_2025.csv  
  → Historical Tesla stock data including price, volume, and technical indicators

- ev_market.csv  
  → Global EV market data including sales, stock, and adoption trends

## Notes
- Data is sourced from Kaggle (originally from Yahoo Finance and IEA datasets)
- Files are cleaned and ready for analysis
- No missing values in core Tesla dataset
